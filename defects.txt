Дефект №1

Название:
Метод contains не обрабатывает значение None в параметре symbol.

Приоритет:
Средний.

Предусловие:
Открыт класс StringUtils.

Шаги воспроизведения:

1. Создать объект StringUtils.
2. Вызвать метод:
   contains("abc", None)

Фактический результат:
Возникает исключение:
TypeError: index() argument 1 must be str, not None

Ожидаемый результат:
Метод должен корректно обработать значение None и вернуть False
либо вывести понятное сообщение об ошибке.
